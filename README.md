Weather App

A modern and responsive weather application built with Vanilla JavaScript (ES6 Modules), HTML, and CSS. The app fetches real-time weather and forecast data from the Open-Meteo API and provides an intuitive user experience with city search, hourly forecasts, and customizable unit preferences.

Features
Real-time weather data
7-day weather forecast
Hourly forecast breakdown
City search with autocomplete suggestions
Temperature, wind speed, humidity, and precipitation details
User preference persistence using Local Storage
Responsive and clean UI
Loading shimmer effects and error handling
Modular JavaScript architecture

Technologies Used
HTML5
CSS3
JavaScript (ES6 Modules)
Open-Meteo Weather API
Open-Meteo Geocoding API
Local Storage API

What I Learned
Working with REST APIs and asynchronous JavaScript
Debouncing search requests
Managing application state
Modular code organization
Dynamic DOM manipulation
Error handling and user feedback
