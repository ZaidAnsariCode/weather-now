<h1>☁️ Weather App</h1>

<p>
A modern and responsive weather application built with Vanilla JavaScript,
providing real-time weather updates, hourly forecasts, and 7-day weather
predictions for cities around the world.
</p>

<h2>✨ Features</h2>
<ul>
  <li>Real-time weather updates</li>
  <li>7-day weather forecast</li>
  <li>Hourly weather forecast</li>
  <li>City search with autocomplete suggestions</li>
  <li>Temperature, humidity, wind speed, and precipitation information</li>
  <li>Customizable unit preferences</li>
  <li>Local Storage support for saving user settings</li>
  <li>Responsive design for desktop and mobile devices</li>
  <li>Loading animations and error handling</li>
</ul>

<h2>🛠️ Technologies Used</h2>
<ul>
  <li>HTML5</li>
  <li>CSS3</li>
  <li>JavaScript (ES6 Modules)</li>
  <li>Open-Meteo Weather API</li>
  <li>Open-Meteo Geocoding API</li>
  <li>Local Storage API</li>
</ul>
